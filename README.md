## 编写及构建步骤
 ```bash
git clone https://github.com/hisey/front-end-doc.git
```
```bash
$ npm i -g docute-cli
```
选择一个空目录
   ```bash
$ docute init ./
```
```bash
$ docute ./
```
打开 http://localhost:8080
