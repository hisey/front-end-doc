                      # UI框架

                      Element Element，一套为开发者、设计师和产品经理准备的基于 Vue 2.0 的桌面端组件库
                      VUX 一个凑合的 Vue.js 移动端 UI 组件库
                      vant 轻量、可靠的移动端 Vue 组件库
                      Tailwind 实用程序优先的CSS框架，用于快速构建自定义设计

                      # 推荐库

                      img-2一个提高图片加载性能和体验的库，懒加载使用 web worker 模糊预览
                      dayjs一****个轻量级类 moment.js API 时间库
                      uppy一个很好看的也很好用的 前端上传库
                      filepond一个小巧的文件上传库
                      awesome-bookmarks一个内容很全的个人收藏夹 beta
                      vue-element-adminA magical vue admin
                      tui-calendar 功能全面的日程安排日历控件，还支持拖拽
                      big.js解决 js 浮点数问题。 主要就是 Big Number 或者小数点温柔
                      cleave.js用于在输入时格式化输入内容（信用卡格式、日期等）
                      FileSaver.js文件下载插件 很多时候下载会有兼容性问题，它能帮你解决这些问题
                      hammer移动端手势库
                      grade一个可以根据你的 图片 调整底色的插件
                      compressorjs 压缩图片神器 text-mask 可以让 input 按照规则输入(如电话,email,日期,信用卡等)，特殊格式 input

                      # Css && 动画

                      animate.css最有名的动画效果库
                      magic.csscss 动画效果库 类似 animate.css
                      direction-reveal根据鼠标进入位置，展现从不同方向 展现 hover 效果
                      laxxx 滚动特效库 轻量级 压缩完 2kb
                      cssfx优雅的 CSS 动画效果，开箱即用
                      Vue2-animateA Vue.js 2.0 port of Animate.css. For use with Vue's built-in transitions

                      # vue

                      Vue.DraggableDnD 拖拽组件 基于 Sortable.js 的 vue 版本
                      vue-content-placeholders 页面龙骨 skeleton
                      vue-content-loader 页面龙骨
                      vuex-persistedstate vuex 数据持久化
                      v-charts 基于 Vue2.x 封装的 Echarts 图表组件
                      vue-amap 基于 Vue 2.x 和高德地图的地图组件
                      vue-baidu-map 基于 VUE 2.x 的百度地图组件
                      vue-ueditor 基于UEditor的富文本
                      vue-meta-info 设置vue 单页面meta info信息

                      # 推荐插件

                      dplayer一款H5播放器
                      xgplayer 西瓜视频播放器
                      localForage-cn 本地储存
                      bignumber 解决 js 浮点数问题。 主要就是 Big Number 或者小数点温柔
                      localResizeIMG 前端图片压缩，兼容IOS，Android，PC、自动按需加载文件
                      pulltorefresh.js 下拉刷新
                      scroll-out滚动效果（滚动视差）的框架，框架大小不到 1KB,使用回调的方式将相关动画元素的属性进行实时分配
                      grade 一个可以根据你的 图片 调整底色的插件
                      photoswipe 图片预览组件，支持移动端
                      chancejs生成随机数据的库
                      stickybitsCSS 的 position: sticky 是一个很有用的设置，但是老的浏览器不支持。这个 JS 库是该功能的垫片库。
                      fuse.js轻量级前端模糊查询库 非常的好用

                      # webpack

                      prerender-spa-plugin vue预渲染，vue 解决seo优化
                      npkill 列出并可以删除无用的 node_modules

                      # node

                      nestjs Nest 是一个用于构建高效，可扩展的 Node.js 服务器端应用程序的框架
