            # UI框架

            Element Element，一套为开发者、设计师和产品经理准备的基于 Vue 2.0 的桌面端组件库
            VUX 一个凑合的 Vue.js 移动端 UI 组件库
            vant 轻量、可靠的移动端 Vue 组件库
            Tailwind 实用程序优先的CSS框架，用于快速构建自定义设计

            # 推荐库

            img-2一个提高图片加载性能和体验的库，懒加载使用 web worker 模糊预览
            dayjs一****个轻量级类 moment.js API 时间库
            uppy一个很好看的也很好用的 前端上传库
            filepond一个小巧的文件上传库
            awesome-bookmarks一个内容很全的个人收藏夹 beta
            vue-element-adminA magical vue admin
            awesome-bookmarks厉害的前端推荐库收藏夹 big.js解决 js 浮点数问题。 主要就是 Big Number 或者小数点温柔 cleave.js用于在输入时格式化输入内容（信用卡格式、日期等） FileSaver.js文件下载插件 很多时候下载会有兼容性问题，它能帮你解决这些问题 hammer移动端手势库 grade一个可以根据你的 图片 调整底色的插件

            # Css && 动画

            animate.css最有名的动画效果库 magic.csscss 动画效果库 类似 animate.css direction-reveal根据鼠标进入位置，展现从不同方向 展现 hover 效果 laxxx 滚动特效库 轻量级 压缩完 2kb cssfx优雅的 CSS 动画效果，开箱即用

            # vue

            Vue.DraggableDnD 拖拽组件 基于 Sortable.js 的 vue 版本 vue-content-placeholders 页面龙骨 skeleton

            dplayer一款H5播放器
            xgplayer 西瓜视频播放器 store 本地储存 bignumber 解决 js 浮点数问题。 主要就是 Big Number 或者小数点温柔
